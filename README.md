## Flask + MongoDB

### Set up

#### Step 1

```js
py -3 -m venv venv
```

#### Step 2

```js
pip3 install -U pip virtualenv
```

#### Step 3

```js
virtualenv --system-site-packages -p python ./venv
```

#### Step 4

```js
pip install Flask
```

### Install pymongo

```js
pip install pymongo
```

### Auto reload

```js
flask --app app.py --debug run
```

### Run docker

```js
http://127.0.0.1:5000/
```

```js
http://localhost:5000/
```
